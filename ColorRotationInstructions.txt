# ColorRotation

This creates a series of images in a specified folder that rotate through the color spectrum. It takes in an image and shifts the colors around a full rotation to be assembled into a gif using a diffent program. There are a few different algorithms for shifting the color of your image, the default algorithm is "colorShiftDiagonal", the others can be accesed by changing the called function in the main. 

File paths will need to be changed to fit the users computer.
